## Introduction

This is an example guide written in MDX format. It demonstrates basic Markdown features and how they should be rendered using `next-mdx-remote` and styled with Tailwind Typography.

## Basic Formatting

You can use standard Markdown syntax:

*   **Bold text**
*   *Italic text*
*   `Inline code`
*   [A link](https://example.com)

## Code Blocks

Here's an example of a JavaScript code block:

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet('World');
```

And a Python code block:

```python
def add(a, b):
  """Adds two numbers."""
  return a + b

result = add(5, 3)
print(f"The result is: {result}") 
```
